(async, static) esploraFetchFeeEstimates(baseUrl) → {Promise.<Object>}

Source:
  • src/dataFetchers.js, line 106

Fetches /fee-estimates from an esplora service.

Get an object where the key is the confirmation target (in number of blocks) and the value is the estimated feerate (in sat/vB).

The available confirmation targets are 1-25, 144, 504 and 1008 blocks.

Parameters:
Name Type Description
baseUrl string

The Base URL of the Esplora server. Defaults to http://127.0.0.1:3002
Returns:

An object where the key is the confirmation target (in number of blocks). For example:

{ "1": 87.882, "2": 87.882, "3": 87.882, "4": 87.882, "5": 81.129, "6": 68.285, ..., "144": 1.027, "504": 1.027, "1008": 1.027 }
Type
Promise.<Object>

(async, static) esploraFetchUtxos(address, baseUrl) → {Promise.<Array>}

Source:
  • src/dataFetchers.js, line 74

Recursively fetches /tx/:txid/hex in an esplora service for all the confirmed utxos of a certain address.

It first fetches all the unspent outputs of an address with '/address/:address/utxo' and then loops over those utxos.

Parameters:
Name Type Description
address string

A Bitcoin address
baseUrl string

The Base URL of the Esplora server. Defaults to http://127.0.0.1:3002
Returns:

An array of utxos objects like this: [{ tx, n },...], where tx is a string in hex format and n is an integer >= 0.

Type
Promise.<Array>
