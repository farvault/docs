Explorer

Explorer

Class representing a client of a blockchain explorer.

Esplora and electrum servers are supported.

Constructor

new Explorer(params)

Source:
  • src/explorer/index.js, line 26

Usage:

  • new Explorer() will use public Blockstream's esplora server.
  • new Explorer({service: ELECTRUM}) will use public Blockstream's electrum server.
  • new Explorer({url: LOCAL_ESPLORA_URL}) will use a esplora server on LOCAL_ESPLORA_URL.

Or choose any other server:

Parameters:
Name Type Description
params object
Properties
Name Type Attributes Default Description
service string <optional>
 ESPLORA

ESPLORA or ELECTRUM. Set url if service = ESPLORA or host, port, protocol if service = ELECTRUM.
url string

Esplora's API url. Defaults to blockstream.info if service = ESPLORA.
host string

Elecrum's host. Defaults to 'electrum.blockstream.info' if service = ELECTRUM and host, port and protocol are unset.
port number

Elecrum's port. Defaults to 50002 if service = ELECTRUM and host, port and protocol are unset.
protocol protocol

Elecrum's protocol. Either 'ssl' or 'tcp'. Defaults to 'ssl' if service = ELECTRUM and host, port and protocol are unset.
network object <optional>
 networks.bitcoin

A network.

(async) fetchFeeEstimates() → {Promise.<Object>}

Source:
  • src/explorer/index.js, line 149

Get an object where the key is the confirmation target (in number of blocks) and the value is the estimated feerate (in sat/vB).

The available confirmation targets are 1-25, 144, 504 and 1008 blocks.

Returns:

An object where the key is the confirmation target (in number of blocks). For example:

{ "1": 87.882, "2": 87.882, "3": 87.882, "4": 87.882, "5": 81.129, "6": 68.285, ..., "144": 1.027, "504": 1.027, "1008": 1.027 }
